def solution(n, k):
    if n > 9:
        return 12000*n-((n//10)*2000) + k*2000
    else :
        return 12000*n + k*2000

# 3. 머쓱이보다 키 큰 사람

[코딩테스트 연습 - 머쓱이보다 키 큰 사람](https://school.programmers.co.kr/learn/courses/30/lessons/120585)

```python
def solution(array, height):
    x = 0
    
    for i in array:
        if i > height:
            x += 1
            
    return x
```

```python
# 다른 풀이

def solution(array, height):
    array.append(height)
    array.sort(reverse=True)
    return array.index(height)
```

# 8. 짝수 홀수 개수

[코딩테스트 연습 - 짝수 홀수 개수](https://school.programmers.co.kr/learn/courses/30/lessons/120824)

```python
def solution(num_list):
    x = 0
    y = 0
    
    for i in num_list:
        if i % 2 !=0:
            x = x + 1
        else:
            y = y + 1
    
    final_list = [y, x]
    
    return final_list
```

```python
# 다른 풀이

def solution(num_list):
    answer = [0,0]
    for n in num_list:
        answer[n%2]+=1
    return answer
```
